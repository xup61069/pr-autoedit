# PR剪教學一條龍 —— 新手指南

這份給「完全沒碰過程式」的人看。照著做就行,看到不懂的字直接跳過。

---

## 它在幫你做什麼

你錄好一支教學影片丟進來,它會自動:

- 把你講話中間的**停頓、廢話、口頭禪**剪掉
- 把**沒講話但有聲音**的地方(播音樂、示範音效)留下來,不會誤剪
- 產生**繁體中文字幕**
- 把剪好的成果**直接送回 Premiere**,你只要看過、微調、輸出

實測一支 17 分鐘的教學片自動剪成 9 分鐘左右,原本兩三小時的剪輯變成幾分鐘看過就好。

**它不會直接幫你輸出成品。** 最後拍板的永遠是你,它只是把無聊的部分先做完。

---

## 你需要準備什麼

| 需要 | 說明 |
|------|------|
| Windows 電腦 | Windows 10 或 11 |
| Premiere Pro | 2024 或更新的版本 |
| NVIDIA 顯卡 | **建議**。沒有也能跑,但語音辨識會慢很多(可能慢 5~10 倍) |
| 硬碟空間 | 約 10GB(語音辨識模型比較大) |
| 網路 | 安裝時要下載東西 |

其他像 Python、ffmpeg 這些名詞你不用懂,安裝程式會自己處理。

---

## 安裝(只做一次)

### 第 1 步:放好資料夾

把下載的壓縮檔解開,整個資料夾放到你想放的地方,例如 `C:\pr-autoedit`。

> ⚠️ 資料夾路徑**不要有中文和空格**,不然某些工具會出錯。
> 例如 `C:\pr-autoedit` 可以,`D:\我的工具\自動剪輯` 不行。

### 第 2 步:雙擊「安裝.bat」

會跳出一個黑色視窗開始跑,大約 **10~30 分鐘**(主要在下載)。
中間可能會出現安裝其他軟體的畫面,讓它跑完就好。

看到 **「安裝完成!」** 就代表好了。

> 如果視窗閃一下就不見,或出現紅色「失敗」的字,看最下面的「遇到問題」。

### 第 3 步:重開 Premiere

把 Premiere **完全關掉**,再重新打開。

然後在上方選單點:**視窗 (Window) > 擴充功能 (Extensions) > PR剪教學一條龍**

面板就會出現了。可以把它拖到旁邊固定住,以後都在。

---

## 每次怎麼用

1. 在 Premiere 開一個專案(新的舊的都行)
2. 面板上按 **「1. 選擇影片」**,挑你錄好的影片檔
3. 按 **「2. 一鍵自動剪輯」**
4. 等它跑完(下面黑框會一直有訊息在動,那是正常的)
5. 跑完會自動把剪好的序列和字幕放進你的專案

**第一次跑會下載語音辨識模型(約 3GB),會比較久,只有第一次。**

### 跑完之後

面板下方會出現「剪輯後工具」:

| 按鈕 | 什麼時候用 |
|------|-----------|
| **開啟審閱報告** | 跑完先按這個。用瀏覽器看哪裡被剪掉、省了多少時間 |
| **重算剪輯** | 覺得剪太兇或太保守 → 改設定 → 按這顆,幾秒就出新版本 |
| **幫目前序列掛人聲處理** | 想把人聲清乾淨、音量壓穩的時候(降噪 + EQ + 壓縮器) |
| **用目前序列產生字幕** | 你自己又剪過之後,讓字幕重新對準 |
| **清除快取** | 平常用不到。只有「結果怪怪的但設定沒動過」時才按 |

## 覺得剪得不對?調這幾個就好

展開面板上的 **「⚙ 設定」**,常用的只有這幾個:

| 設定 | 白話說明 |
|------|---------|
| **教學類型** | 選你影片的領域(motion/PV/剪輯/編曲/特效/3D/動畫/遊戲/程式/攝影)。**這個最有效**,選對了專有名詞才不會被聽錯。內建的詞不夠用?按旁邊的「✎ 編輯類型」自己加,也可以開自己的新類型 |
| **我的額外術語** | 就在教學類型正下方。頻道名、人名、你的慣用詞填這裡,逗號分隔。**這一欄永遠不會被砍掉**,最重要的詞放這邊最保險 |
| **停頓多久才算靜音** | 數字調小 = 剪比較兇。講話慢的人調大一點 |
| **停頓怎麼處理** | 三選一。**看畫面決定**(建議):你默默示範操作的那幾秒畫面在動,會快轉帶過留下來;真的在發呆才剪掉。**一律快轉**:什麼都不刪,只是壓縮。**一律剪掉**:最兇,但默默示範的畫面也會不見 |
| **能量微剪** | 打勾 = 連講話中間的小空檔也剪掉,可以再省 2~3 分鐘 |
| **保護音樂/音效段** | 保持打勾。這樣你播的示範音樂才不會被剪掉 |
| **剪掉短促雜音** | 保持打勾。咳嗽、清喉嚨、滑鼠喀一聲會被剪掉。**講話講到一半咳的剪不掉**(聲音疊在一起分不開),那種要自己處理 |

改完按 **「重算剪輯」**,幾秒鐘就出一個新序列,舊的還在,不喜歡就刪掉。
**隨便試沒關係,不會弄壞任何東西。**

## 遇到問題

**面板選單裡找不到**
→ Premiere 有沒有完全關掉再開?再跑一次「安裝.bat」試試。

**按下去說找不到 Python**
→ 重新跑一次「安裝.bat」。如果還是不行,可能是 Python 沒裝好,
   到 https://www.python.org/downloads/ 自己裝一次,
   **安裝第一頁一定要勾「Add python.exe to PATH」**,裝完再跑「安裝.bat」。

**跑到一半失敗了**
→ 看面板下方黑框裡**紅色**的字,那是錯誤訊息。把它整段複製下來問人或查一下。

**辨識出來的字很多錯**
→ 到設定裡選對「教學類型」,再把你常講的專有名詞、頻道名填進「我的額外術語」。
   改完下次跑會自動重新辨識。

**剪太兇,話被切掉**
→ 設定裡把「停頓多久才算靜音」調大,或把「能量微剪」的
   「每個停頓留多少」調大(例如從 0.06 改成 0.1),再按「重算剪輯」。

**想整個移除**
→ 雙擊「移除面板.bat」。你的影片和剪輯成果都不會被刪。
