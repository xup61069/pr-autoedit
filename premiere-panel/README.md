# PR 自動剪輯 —— Premiere 面板(CEP)

在 Premiere Pro 裡按一個按鈕,就跑完整條自動剪輯,並把剪好的序列與字幕匯回來。
你完全不用離開 Premiere。

> 這是「薄薄一層」的面板:真正的功能(降噪、轉錄、剪輯)都在上層的 Python
> 程式裡。面板只負責「啟動 Python」和「匯入結果」。

## 運作方式

```
面板按鈕 → 執行 ../pipeline.py 你的影片 → 匯入 output/影片名/04_project.xml + 字幕
```

## 安裝(只需做一次)

1. **確認路徑**:打開 `js/main.js`,檢查最上面兩行是否符合你的電腦:
   ```js
   var PROJECT_DIR = "C:\\pr-autoedit";                         // 專案資料夾
   var PYTHON      = "C:\\Users\\Administrator\\miniconda3\\python.exe"; // Python
   ```

2. **開啟開發者模式**:雙擊執行 `enable-debug-mode.reg`(允許載入自製面板)。

3. **安裝面板**:雙擊執行 `install.bat`(把本資料夾連結到 Premiere 擴充目錄)。

4. **重啟 Premiere Pro**,然後在選單:
   `視窗 (Window) > 擴充功能 (Extensions) > PR 自動剪輯`

## 使用

1. 先在 Premiere 開啟(或新建)一個專案。
2. 面板上按「選擇影片」挑你的錄影檔。
3. (可選)展開「⚙ 設定」調參數,見下。不調也能跑,預設值就能用。
4. 按「一鍵自動剪輯」,等進度跑完(狀態列黃色=處理中、綠=完成、紅=出錯;
   下方黑框是詳細訊息,錯誤行會標紅)。
5. 完成後會自動匯入剪好的序列與字幕,並自動切到那條新序列。

> **重跑同一支影片會「換掉」上次那條序列**,專案裡不會愈堆愈多。
> 序列名稱帶影片名(例:`我的教學 自動剪輯`),所以只會換掉這支片自己的舊序列,
> 別支影片的序列不受影響。想保留兩個版本互相比較,請改用「重算剪輯」那顆
> ——它刻意保留舊序列。

## 剪輯後工具(跑完出現,重開面板也還在)

- **開啟審閱報告**:瀏覽器打開這次剪輯的報告——哪裡被剪、哪裡被快轉、
  省了多少時間、音樂段抓在哪,一分鐘掃完。
- **重算剪輯 → 匯入新序列**:改了任何設定(停頓門檻、快轉倍率、交付方式…)
  後按這顆。不重跑辨識,幾秒重算並匯入一個**新**序列;舊序列不動,隨時可反悔。
- **幫目前序列掛降噪(實驗)**:把降噪外掛掛到序列的每個聲音片段
  (音樂段除外),之後在「效果控制」隨時調。失敗就改走穩定做法:
  視窗 > 音軌混音器,A1 軌效果插槽選 VoiceFX,整軌生效。
- **用目前序列產生字幕**:在 Premiere 剪完後按這顆,字幕依時間軸目前的
  實際樣子重新對位(刪掉的內容自動消失、快轉段自動壓縮)並匯入新的 SRT。

## ⚙ 設定(面板內直接調)

展開「⚙ 設定」會自動載入所有可調參數,分兩頁:

- **基本頁**(分組:辨識、剪輯):教學類型(選領域自動載入術語,可複選)、
  辨識引擎/模型/語言、交付方式(baked 直接剪好 / live 全保留+標籤)、
  停頓怎麼處理、快轉倍率、靜音門檻、音樂保護開關。
- **進階頁**(平常不用動,每項有白話說明):降噪外掛(VST)、
  冗詞與口頭禪、字幕、審閱標記、音樂偵測靈敏度、其他微調。

小技巧:數字滑條**點兩下恢復預設值**;改了辨識相關設定(引擎/模型/詞庫)
不用清快取,下次剪輯會自動重新辨識。

按「儲存設定」會寫入 `config\settings_local.json`(不動共用設定、更新專案不會被蓋)。
按「一鍵自動剪輯」時也會自動先存一次目前的設定。

> 欄位是由 `..\ui_settings.py` 自動產生的;之後專案新增參數,面板會自動長出對應控制項。

## 疑難排解

- **面板選單裡找不到**:確認做了步驟 2(reg)和 3(install.bat),並已重啟 Premiere。
- **按下去說找不到 Python**:檢查 `main.js` 的 `PYTHON` 路徑。
- **想看詳細錯誤**:面板下方的黑框會顯示 Python 的完整輸出訊息。
- **面板改版後沒更新**:因為用 junction 連結,改完存檔重開面板即可;
  改 `manifest.xml` 則要重啟 Premiere。
